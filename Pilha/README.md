
# Pilha (Stack)

## Conceito LIFO (Last-In, First-Out)

Em uma estrutura de dados do tipo pilha, o último elemento a ser inserido é o primeiro a ser removido. É como uma pilha de pratos, onde o último prato que você colocou em cima é o primeiro que você retira.

### Operações Básicas:

*   **Push (Empilhar):** Adiciona um elemento ao topo da pilha.
*   **Pop (Desempilhar):** Remove o elemento do topo da pilha.
*   **Peek (Espiar):** Retorna o elemento do topo da pilha, sem removê-lo.
*   **isEmpty (Está Vazia):** Verifica se a pilha está vazia.
*   **isFull (Está Cheia):** Verifica se a pilha está cheia.
