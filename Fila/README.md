
# Fila (Queue)

## Conceito FIFO (First-In, First-Out)

Em uma estrutura de dados do tipo fila, o primeiro elemento a ser inserido é o primeiro a ser removido. É como uma fila de supermercado, onde a primeira pessoa a chegar é a primeira a ser atendida.

### Operações Básicas:

*   **Enqueue (Enfileirar):** Adiciona um elemento ao final da fila.
*   **Dequeue (Desenfileirar):** Remove o elemento do início da fila.
*   **Peek (Espiar):** Retorna o primeiro elemento da fila, sem removê-lo.
*   **isEmpty (Está Vazia):** Verifica se a fila está vazia.
*   **isFull (Está Cheia):** Verifica se a fila está cheia.
